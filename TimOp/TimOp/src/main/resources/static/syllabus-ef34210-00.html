교과목
개요		 본 교과목은 인공지능(AI)의 기초 이론과 응용을 학습하는 것을 목표로 한다. 인공지능의 역사와 발전 배경을 이해하고, 머신러닝과 딥러닝의 기본 원리와 실습을 통해 AI 기술의 응용 능력을 배양한다. 최신 AI 기술인 생성형 언어 모델(GPT), 생성 모델(GAN, Stable Diffusion), 그리고 강화 학습을 학습하여 이론적 이해와 실무적 활용 역량을 동시에 갖추도록 설계되었다.
교과목
운영		 수업목표 1. 인공지능과 머신러닝의 기본 개념과 역사를 이해하고, 실생활에서의 활용 사례를 탐구한다.
2. 파이썬과 주요 라이브러리를 활용하여 데이터 처리, 분석, 시각화, 그리고 딥러닝 모델 구현 능력을 배양한다.
3. 머신러닝과 딥러닝의 핵심 알고리즘(선형 회귀, 퍼셉트론, MLP, CNN, RNN 등)을 이론적으로 이해하고 실습을 통해 구현 능력을 갖춘다.
4. 케라스와 같은 딥러닝 라이브러리를 활용하여 다양한 실제 문제(이미지 분류, 자연어 처리, 영상 인식 등)를 해결하는 경험을 쌓는다.
